# friends

친구기능 rn 레포지토리

## installation

```bash
git clone https://github.com/wafflestudio/snutt-frontend
cd snutt-frontend/apps/friends-react-native
yarn
```

.env.local 을 셋업합니다.

```env
API_URL=https://snutt-api-dev.wafflestudio.com
ASSET_URL=http://localhost:8082
```

## run

```bash
yarn start
yarn start:assets
```
